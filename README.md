# 👥 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a modern web application designed to help you maintain meaningful relationships.  
It tracks your interactions with friends and reminds you when it’s time to reconnect.

---

## Live Demo

🔗 https://your-live-link.com

---

## Project Overview

In today’s busy world, it’s easy to lose touch with friends.  
KeenKeeper helps you stay connected by tracking communication history and reminding you to reach out at the right time.

---

## Technologies Used

- React.js / Next.js
- Tailwind CSS
- React Router DOM / App Router
- Recharts (for analytics)
- React Toastify (for notifications)

---

## Key Features

### 1. Friend Management

- View all friends in a responsive card layout
- Track contact status (Overdue / Almost Due / On Track)
- Detailed friend profile page with full information

### 2. Quick Check-In System

- Instantly log interactions (Call, Text, Video)
- Automatically updates timeline with current date
- Shows real-time toast notifications

### 3. Friendship Analytics

- Visual pie chart of interactions (Call / Text / Video)
- Helps understand communication patterns

---

## Pages Included

- Home Page (Friends + Summary)
- Friend Details Page
- Timeline Page (Interaction History)
- Stats Page (Analytics)
- 404 Not Found Page
